Virtual Machine

1. Registers

The virtual machine has 4 registers: pc, sp, r, and flags.

pc is the program counter. It refers to the offset in the executable file which is being executed. This register can only be modified indirectly through the branch instruction.

sp is the stack pointer. It points to the current cell in memory.

r is the general purpose register. It is 1 byte in size.

1.1 Flags Register

The flags register is a 4-bit register which contains the following bit flags:

Bit Flag Description
0 Z Zero flag
1 N Negative flag
2 C Carry flag
3 V Overflow flag

2. Memory

Memory is organized into a finite series of 1 byte cells. The lowest address is 0. sp is initialized to 0 when a program starts.

Branch instruction

The branch instruction sets pc to the given address if the condition is true.

After the opcode, the instruction expects a nybble condition code, followed by an address. The below table describes the nybble condition code. The low 3 bits' value are determined by the condition table. The high bit, called the negate bit, dictates whether the condition should be negated after being evaluated.
