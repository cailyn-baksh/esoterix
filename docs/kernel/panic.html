Kernel Panic

A kernel panic occurs when a fatal error occurs which the kernel is unable to recover from. A kernel panic will print a generic message and an error code to UART1: 

Kernel panic. Code xx

Where xx is a hexadecimal error code. If an error occurs during the panic, or if some other factor prevents the error message from printing, the error code is also stored in register r10, which is guaranteed to not be overwritten at any point during a kernel panic.
